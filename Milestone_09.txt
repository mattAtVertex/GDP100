GDP Milestone 9: Your First Game Lesson Plan
Objective

Students will learn how to import the base level, polish the combat system, implement advanced movement, create enemies, set up pick-ups, and upgrades in a game using Unreal Engine. Students will be assessed on their ability to import the base level, polish the combat system, implement advanced movement, create enemies, set up pick-ups, and upgrades in a game. They will also be evaluated on their ability to work collaboratively with their peers, follow instructions, and problem-solve independently.

Lessons

Importing the Base Level
Combat System Polish
Advanced Movement
Enemy Creation 
Pick Up and Upgrades
